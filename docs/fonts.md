# Fonts

This starter assumes a Nerd Font for prompt glyphs and file icons.

## Recommended

- `JetBrainsMono Nerd Font` for the main terminal font
- `Symbols Nerd Font Mono` as a fallback if you want extra icon coverage

## Ghostty

Open Ghostty settings and use:

```text
font-family = JetBrainsMono Nerd Font
```

## Terminal.app

If you also use Terminal.app, switch its profile font to a Nerd Font too. Otherwise `agnoster` glyphs may render as boxes.
